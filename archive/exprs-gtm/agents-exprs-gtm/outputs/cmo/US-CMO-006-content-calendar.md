## Publishing Cadence Rules

1. **Whitepapers publish before blogs that reference them.** No blog should cite a whitepaper that is not yet available.
2. **CRO emails reference published blogs.** Every email that links to content must link to live, accessible pages.
3. **Social posts extract from published blogs.** Social content is derivative, not original -- maintains consistency.
4. **Gates precede dependent work.** POSITION-READY before CRO sequences. SEQUENCE-DRAFT before Phase 2 expansion.
5. **One blog per week maximum.** Quality over frequency. Each post must meet brand voice standards.
6. **All blog data claims cite Harvard-format sources.** Every numerical claim in a blog post must reference its source using (Author, Year) in-text format per CITATION-STANDARD.md.
