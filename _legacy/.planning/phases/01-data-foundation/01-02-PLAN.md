# Plan 01-02: CSV Upload & Mapping Interface

## Goal

Build a Python-based CSV upload and data merge system that imports Apollo contact exports (and other CSV sources) into Supabase, preserving critical enrichment fields, implementing smart merge logic (new fields fill blanks, existing data preserved), and tracking data provenance. This enables manual enrichment of the 10 pilot companies without API costs.

## Must-haves

1. **CSV parser** that handles Apollo export format with all critical fields (contact: name, email, title, seniority, departments; company: name, domain, industry, employee_count, funding, revenue)
2. **Field mapping interface** (which CSV columns → which Supabase table fields) with configurable mapping profiles
3. **Data merge logic** that preserves existing data and fills blanks with new data (not full overwrite)
4. **Validation engine** that checks data quality (required fields, email format, domain format, duplicate detection)
5. **Multi-table distribution** (single CSV row → companies table + prospects table, linked correctly)
6. **Source tracking** for each upload (timestamp, filename, row count, fields mapped)
7. **Error handling** with detailed logging (which rows failed, why, partial success support)
8. **Candidate CSV support** (separate format for supply-side talent uploads)
9. **CLI interface** for running uploads (with dry-run mode)
10. **Documentation** with example CSVs and field mapping guide

def merge_company_data(existing_company, new_data):
    """
    Merge strategy: New data fills blanks, existing data preserved

    Rules:
    - If existing field is NULL and new field has value → use new value
    - If existing field has value and new field has value → keep existing (preserve truth)
    - If both NULL → remain NULL
    - Always update updated_at timestamp
    - Track which fields were updated in metadata
    """
    merged = existing_company.copy()
    updated_fields = []

    for field, new_value in new_data.items():
        if field in ['id', 'created_at', 'source_of_truth']:
            continue  # Never overwrite these

        existing_value = merged.get(field)

        if existing_value is None and new_value is not None:
            merged[field] = new_value
            updated_fields.append(field)

    # Update metadata to track merge
    if 'metadata' not in merged:
        merged['metadata'] = {}
    merged['metadata']['last_merge'] = {
        'timestamp': datetime.now().isoformat(),
        'fields_updated': updated_fields,
        'source': new_data.get('source_of_truth', 'unknown')
    }

import typer
from supabase import create_client, Client
from pathlib import Path
import logging

app = typer.Typer()

class CSVUploader:
    def __init__(self, supabase_client: Client):
        self.client = supabase_client
        self.logger = logging.getLogger(__name__)

    def upload_apollo_csv(self, filepath: Path, dry_run: bool = False):
        """
        Main upload flow:
        1. Parse CSV with Apollo mapping
        2. Validate all rows
        3. For each row:
           a. Extract company data
           b. Upsert company (merge if exists)
           c. Extract prospect data
           d. Link prospect to company
           e. Upsert prospect (merge if exists)
        4. Track source (CSV filename, timestamp)
        5. Return upload summary
        """
        # Parse CSV
        rows = parse_csv(filepath, mapping_profile='apollo')

        # Validate
        valid_count, error_count, errors = validate_csv_batch(rows, 'prospect')

        if dry_run:
            return self._dry_run_report(rows, valid_count, error_count, errors)

        # Upload
        uploaded_companies = []
        uploaded_prospects = []
        failed_rows = []

        for idx, row in enumerate(rows):
            try:
                # Upsert company
                company = self._upsert_company(row['company_data'])
                uploaded_companies.append(company['id'])

                # Upsert prospect
                prospect = self._upsert_prospect(row['prospect_data'], company['id'])
                uploaded_prospects.append(prospect['id'])

            except Exception as e:
                self.logger.error(f"Row {idx} failed: {e}")
                failed_rows.append({'row': idx, 'error': str(e)})

        return {
            'companies_processed': len(uploaded_companies),
            'prospects_processed': len(uploaded_prospects),
            'failed_rows': failed_rows,
            'source': filepath.name,
            'timestamp': datetime.now().isoformat()
        }

    def upload_candidate_csv(self, filepath: Path, dry_run: bool = False):
