# Legacy / Archived Files

These files originated from the parent **AI Security Recruitment Sales Intelligence Platform** project and do not belong in the `shared-ai-layer` repository, which is a backend AI orchestration layer.

They have been moved here to isolate them from the active codebase during refactoring. They can be:
- Migrated to a separate repository for the parent platform
- Deleted once no longer needed for reference

## Contents

| Directory | Description |
|-----------|-------------|
| `backend/` | Python modules for RSS aggregation, CSV upload, signal classification, company matching, etc. |
| `docs/` | Documentation for CSV uploads, RSS feeds, and the parent platform's database schema |
| `tests/` | Tests for CSV upload and RSS processing pipelines |
| `.planning/` | Product roadmap, requirements, and phase plans for the parent platform |
| `audit/` | Company audit templates, checklists, and examples |
| `ralph/` | Ralph agent prompt, launch script, PRD, and progress logs |
| `archive/` | Historical snapshots of earlier PRD and progress files |
| `check-openai.js` | One-off debug script for checking OpenAI constructor types |
