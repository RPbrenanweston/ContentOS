# Plan 03-02: Learning Loop + Prompt Updates

## Goal

Implement the agent learning mechanism that analyzes feedback patterns weekly and updates agent system prompts to improve performance based on real user ratings.

## Must-haves

1. **Feedback Pattern Analysis**
   - Weekly job analyzes all feedback from past 7 days
   - Identifies high-performing vs low-performing prompt patterns
   - Extracts common characteristics of thumbs-up vs thumbs-down outputs
   - Generates actionable insights (e.g., "shorter emails get +15% approval", "company context in first line +22% approval")

2. **Prompt Version Management**
   - Version control system for agent prompts (v1.0, v1.1, etc.)
   - Store prompt history with metadata (created_at, performance_stats, rollback_capability)
   - A/B testing framework (run 2 prompt versions simultaneously)
   - Automatic version increment on updates

3. **Prompt Update Logic**
   - Analyze top 20% performing outputs vs bottom 20%
   - Identify structural differences (length, tone, formatting, content order)
   - Generate updated prompt suggestions based on patterns
   - Apply updates to agent system prompts
   - Maintain prompt version in agent outputs for tracking

4. **Learning Cycle Execution**
   - Automated weekly job (cron or scheduled task)
   - Runs analysis → generates insights → updates prompts → logs results
   - Sends summary report (email or dashboard notification)
   - Fallback/rollback if new prompt performs worse

5. **Performance Tracking**
   - Compare prompt version N vs N-1 performance
   - Track approval rate by prompt version
   - Monitor improvement trend over time
   - Flag degradation and auto-rollback

  for (const agentType of agentTypes) {
    console.log(`\n=== Analyzing ${agentType} ===`);

    // Step 1: Analyze feedback
    const insights = await analyzeWeeklyFeedback(agentType, startDate, endDate);

    // Step 2: Store insights in database
    await storeLearningInsights(agentType, startDate, endDate, insights);

    const shouldUpdate = insights.approval_rate < 0.7 || insights.recommendations.length > 2;

export class AgentPromptConfig {
  private cache: Map<AgentType, string> = new Map();

  async getPrompt(agentType: AgentType): Promise<string> {
    // Check cache first (TTL: 1 hour)
    if (this.cache.has(agentType)) {
      return this.cache.get(agentType)!;
    }

    // Fetch active prompt from database
    const promptVersion = await promptVersionManager.getCurrentPrompt(agentType);
    this.cache.set(agentType, promptVersion.prompt_text);

    return promptVersion.prompt_text;
  }

  clearCache() {
    this.cache.clear();
  }
}

export const agentPromptConfig = new AgentPromptConfig();
